Flexbox helps groups of items line up, share space, and wrap when needed.

Flexbox starts on the parent:

Flexbox is a parent-child layout mode. You put display: flex on a parent element, and the direct children of that parent become flex items.

By default, when you apply display: flex on an element, flexbox arranges the direct children in a row. That row is called the main axis. The opposite direction is called the cross axis.

Use Gap For Button Rows:

The filter buttons are a group. The space between them should belong to the group, so gap is a better fit than random margins on each button.

Add Flexbox to .toolbar

display: flex makes the buttons flex items.

gap: 8px creates consistent space between them.

flex-wrap: wrap lets items move onto a new line when there is not enough horizontal space. Without wrapping, Flexbox tries to keep items on one line, which can make a row feel cramped or overflow on smaller screens.

Let Cards Share Available Space:

.lesson-list is the parent, so its direct children, the cards, become flex items.

flex: 1 tells each card it can grow and share the available space with the other cards. For now, read it as "let this item take an equal share of the row."

min-width: 220px protects each card from becoming too narrow. When there is not enough room for all cards to stay at least 220px wide, flex-wrap: wrap lets the next card move to a new line.

margin-bottom: 0 removes the old vertical spacing from the stacked version. Once the cards are in a flex group, gap owns the spacing between them.


          .lesson-list {
            display: flex;
            gap: 16px;
            flex-wrap: wrap;
            margin-top: 24px;
          }
          .lesson-card {
            flex: 1;
            min-width: 220px;
            margin-bottom: 0;
          }

Align small rows inside a card:

Flexbox is not only for big page sections. It is also useful for small rows inside components.

Each card has a footer with two direct children.

The time label and the link are direct children of .card-footer, so .card-footer is the parent that needs Flexbox.

justify-content: space-between pushes the time label and link apart

align-items: center lines them up vertically

ap: 12px keeps a little space between them if the row gets tight.
