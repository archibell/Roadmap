What the server actually receives:

Produces a request like:

text:
GET /search?q=html&page=2 HTTP/1.1

The same form switched to method="post":

Produces a request like:

text:
POST /search HTTP/1.1
Host: example.com
Content-Type: application/x-www-form-urlencoded
